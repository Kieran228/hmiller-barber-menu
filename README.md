# Barber Service Menu

A modern, mobile-friendly barber service menu designed to clearly showcase services, pricing, and add-ons. Built with a clean UI and smooth user experience, this menu helps barbers present their offerings professionally on any device.

## Features
- Mobile-first, responsive design  
- Clean and modern layout  
- Easy-to-read service and pricing structure  
- Fast-loading and lightweight  
- Simple to customize for any barbershop brand  

## Use Cases
- Barbershop websites  
- In-shop digital menus  
- Link-in-bio or QR code menus  
- Personal barber portfolios  

## Customization
This menu is designed to be easily customized with your own services, prices, colors, and branding. No unnecessary complexity - just update the content and make it yours.
